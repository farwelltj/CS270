Javascript Reserved Words

Break - Terminates the current loop,switch,or label statement and transfers program control to the statement following the terminated statement.

Switch - Evaluates and expression, matching the expression's value to a case label, and executes statements associated with that case.

Catch - Marks a block statements to try, and specifies a reponse, should an exception be thrown.

Continue - Terminates execution of statements in the current iteration of the current or labelled loop, and continues execution of the loop with the next iteration.

Debugger - Invokes any available debugging functionally. If no debuggin functionality is available, this statment has no effect.

Switch - Evaluates an expression, matching the expression's vaule to a case label, and executes statements associated with that case.

Delete - Unlike what comon beliefs suggests, the delete operator has noting to do with directly freeing memory(it only does indirectly via breaking references)

While - Creates a loop that executes a specified statment until the test condition evaluates to false. The condition is evaluated after executing the statement, resulting in the specified statment executing at least once.

Else - Executes a statement if specified condition is true. If the condition is false, another statement can be executed.

Catch - Marks a block of statements to try, and specifies a response, should an exception be thrown.

For - Creates a loop that consists of three optional expressions, enclosed in parentheses and separated by semicolons, followed by a statement executed in the loop.

Function - Declares a function with the specified parameters.

In - Iterates over the enumerable properties of an object, in arbitary order. For each distinct property, statements can be executed.

Instanceof - The instanceof operator tests whether an object has in its prototype chain the protoype of a constructor.

New - The new operator creates an instance of a user-defined object type or of one of the built-in object types that has a constructor function.

Return - Specifies the value to be returned bya funciton

This - A function's this keyword behaves a little differently in JavaScript compared to other languages. It also has some differences between strict mode and non-strict mode.

Throw - Throws a user-defined exception.

Typeof - The typeof operator returns a string indicating the type of the unevaluated operand.

Var - Declares a variable, optionally initializing it to a value.

Void - The void operator evaluates the given expression and then underfined

With - Extends the scope chain for a statement
